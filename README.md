# 南财校园网自动登录工具

> Windows 下南京财经大学校园网**一键自动登录** | NUFE Campus Network Auto-Login for Windows

---

## ✨ 功能特性

- 🔄 **开机自动登录**：登录 Windows 桌面后在后台自动完成校园网认证，无需手动操作
- 🔁 **断线自动重连**：后台持续监测网络状态，掉线后自动重新认证
- 🌐 **多运营商支持**：支持选择中国移动、中国电信等运营商
- 🖱️ **图形化配置界面**：首次配置通过 GUI 弹窗完成，简单直观
- 🔒 **安全存储密码**：密码加密保存为当前 Windows 用户专属凭据，其他账户无法读取
- 📋 **日志查看**：一键查看完整的运行记录与登录历史

## 🚀 快速开始

### 系统要求

- Windows 10 / Windows 11
- PowerShell（系统自带，无需额外安装）
- 连接至南京财经大学校园网

### 首次安装与配置

1. [下载](https://github.com/lazymiaomiaomiao/nufe-campus-login/releases) 并解压所有文件到任意目录（建议放在固定位置，如 `C:\tools\nufe-login\`）
2. 双击 **`一键部署.cmd`**
3. 在弹出的 GUI 窗口中：
   - 选择您的**运营商**（移动 / 电信）
   - 填写您的**校园网账号**（通常为学号）
   - 填写您的**密码**
4. 点击 **"保存并部署"**，程序将自动配置开机自启动

✅ 配置完成后，每次开机将自动在后台登录校园网。

## 🔧 工作原理

本工具完全基于 **Windows PowerShell** 和 **.NET WinForms** 开发，无需安装任何第三方依赖：

1. `NufeCampusLogin.ps1` 向南财校园网认证服务器发起 HTTP 登录请求，模拟浏览器认证流程
2. `NufeCampusMonitor.ps1` 定期检测网络连通性，断线时自动调用登录脚本重连
3. 通过 Windows **任务计划程序**（Task Scheduler）实现开机自启动与后台持久运行
4. 密码通过 PowerShell 的 `SecureString` + Windows 数据保护 API（DPAPI）加密存储，仅当前用户可解密

## ❓ 常见问题

**Q：配置完后登录不上怎么办？**  
A：双击 `查看日志.cmd` 查看错误信息；确认账号密码和运营商选择无误；确保当前已连接南财校园内网。

**Q：换宿舍或换网口后还能用吗？**  
A：可以，只要连接的是南财校园网，认证服务器地址不变，工具即可正常使用。

**Q：毕业后如何彻底卸载？**  
A：双击 `卸载自启动.cmd` 移除开机自启任务，然后直接删除整个程序文件夹即可。

**Q：是否会保存我的明文密码？**  
A：不会。密码通过 Windows DPAPI 加密后存储，仅在您自己的 Windows 账户下才能解密，换账户登录或重装系统后密码将失效，需重新配置。
