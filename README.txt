NUFE 校园网登录工具

首次使用：
1. 双击“一键部署.cmd”。
2. 在弹出的窗口里选择运营商，填写账号和密码。
3. 点击“保存并部署”。

日常使用：
- 正常情况下不用手动操作，登录 Windows 桌面后会自动在后台检查。
- 掉线时可以双击“一键重登.cmd”立即重连。
- 需要改账号、密码或运营商时，双击“修改配置.cmd”。
- 需要看运行记录时，双击“查看日志.cmd”。
- 需要关闭开机自启时，双击“卸载自启动.cmd”。

说明：
- 程序只依赖 Windows 自带 PowerShell 和 .NET WinForms，Win10/Win11 默认可用。
- 密码保存为当前 Windows 用户可解密的加密文件，不以明文保存。
- 登录请求直接访问校园网认证 IP，并绕过系统代理，适合本机开代理或 DNS 异常时使用。
- 后台默认每 5 分钟检查一次。正在使用其他 Wi-Fi 时不会自动切换网络；只有断开 Wi-Fi 且附近能扫到 NUFE-STU，才会尝试自动连回校园网。
